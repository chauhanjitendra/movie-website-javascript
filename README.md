# 🎬 Movie Website (JavaScript)

A sleek and dynamic **Movie Website** built using **HTML**, **CSS**, and **JavaScript**.  
This project allows users to explore movies, view posters, and get quick details — all through a clean and modern UI! 🍿

Perfect for film lovers and web dev learners who want to practice working with APIs, DOM manipulation, and responsive design.
## 🌐 Live Demo

🔗 **Check it out live**  
👉 (/https://chauhanjitendra.github.io/movie-website-javascript/)
## 🛠️ Tech Stack

- 🧱 **HTML5** – Structure  
- 🎨 **CSS3** – Styling  
- ⚙️ **JavaScript** – Dynamic functionality & API calls  
- 📡 (Optional) **TMDB API** – For real movie data

---

## ✨ Features

- 🔍 **Search Functionality** – Find your favorite movies instantly  
- 🖼️ **Movie Posters** – Beautifully displayed in a grid layout  
- 📋 **Movie Details** – Title, rating, release date, and overview  
- 🎨 **Responsive Design** – Works on desktop, tablet, and mobile  
- ⚡ **Dynamic Content** – Loaded using JavaScript
